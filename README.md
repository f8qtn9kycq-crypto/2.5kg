# 🎯 60天「可持續性」生活化減重工具

> **目標：** 2個月穩健減重 2.5 kg  
> **理念：** 沒有絕對不能吃的食物，只有不會調配的份量

一個以營養學原則與行為心理學設計的互動式減重追蹤工具。透過階梯式的60天行動計劃，幫助用戶在日常生活中建立可持續的健康習慣。

---

## ✨ 核心特色

### 📊 智能儀表板 (Dashboard)
- **即時追蹤：** 記錄起始體重、目前體重，實時顯示進度百分比
- **進度視覺化：** 漸層進度條，動態反映完成情況
- **營養原則：** 展示營養師的四大核心執行原則

### 📅 60天階梯式行動表
分為4個階段，每階段15天，循序漸進：

- **Phase 1 習慣建立（D1-15）**
  - 調整定錨、引入優質蛋白質、減少碳水化合物
  - 建立基準與觀察無意識進食行為

- **Phase 2 赤字穩定（D16-30）**
  - 精緻澱粉管理、外食防禦策略
  - 建立穩定的300~400大卡溫和赤字

- **Phase 3 體態精雕（D31-45）**
  - 優化食物選擇、強化運動效能
  - 微調進食順序與份量控制

- **Phase 4 最後衝刺（D46-60）**
  - 鞏固習慣、衝刺最後目標
  - 最終檢視與成果確認

### 🚦 應變紅綠燈
針對三大常見情景的行為防禦指南：

1. **突發的外食社交聚餐**
   - 🔴 紅燈：禁止行為
   - 🟡 黃燈：替代方案
   - 🟢 綠燈：首選定錨

2. **工作太累、沒體力運動**
   - 從代償性進食轉向高質量睡眠修復

3. **下午茶時間想吃甜點**
   - 時機防禦與額度管理策略

## 🥗 四大核心執行原則

### ⚖️ 彈性熱量赤字與食物平反
- 維持 300~400 大卡溫和赤字
- 保留心靈慰藉，拒絕極端飲食法

### 🍰 甜點的「時機防禦」規則
- 單次上限 300 大卡以內
- 優先安排在重訓日（運動後肌肉易儲存）
- 或高壓工作日（調節壓力，預防暴食）

### 🍷 酒精的「當量與防禦」規則
- 女性每日上限 1 個酒精當量（啤酒330ml/紅酒120ml）
- **絕不空腹喝** - 飲酒前必先補充高蛋白質與蔬菜定錨血糖

### 🔍 外食的「定錨效應」流程
1. 點餐時眼睛先看「蛋白質」與「蔬菜」
2. 當這兩者分量足夠帶來飽足感
3. 大腦才有理性分配剩下的甜點或酒額度

## 🚀 使用方式

### 基本功能
1. **輸入體重資料**
   - 在儀表板上設定起始體重與目前體重
   - 系統自動計算進度

2. **追蹤每日任務**
   - 切換至「行動表」分頁
   - 選擇相應階段（Phase 1-4）
   - 勾選完成的每日任務

3. **查閱應變策略**
   - 切換至「紅綠燈」分頁
   - 根據當前情況查閱防禦方案

### 數據持久化
- 所有進度與體重記錄存儲於瀏覽器 LocalStorage
- 關閉頁面後數據自動保留
- 支持跨設備同步（需手動輸入）

## 🔧 自訂與擴展

### 修改目標減重量
編輯 `index.html` 中的儀表板卡片：
```html
<div class="metric-card">
    <h3>目標減重</h3>
    <div class="metric-value" style="color: var(--success);">-X.X kg</div>
</div>
```

### 新增行動任務
在 `rawData` 陣列中添加新的行動項目：
```javascript
["日期", "飲食任務", "心靈任務", "動態任務", "筆記"]
```

### 修改階段結構
調整 `phase-btn` 與 `rawData` 的分段邏輯（預設每階段15天）
